1. html구조 Map

html 구조는 lnb 존재에 따라서 2가지 타입으로 구분됨.

2. 파일 / 폴더 설명

  • css - css 파일폴더

  • html - html 파일폴더

    html 폴더에 세가지 폴더가 있음.

    개발할떄 include 부분과 실제 컨텐츠의 영역으로 나누어서 작업하기 쉽도록 가장 기본적인 영역 구분이 되어 있음

    • content - 각 페이지의 [section id="contents"] 영역소스의 파일들 포함하는 폴더

    • include - include 될 수 있는 영역소스의 파일들을 포함하는 폴더

      • footer.html - [div id="bottom_nav"], [footer] 영역 소스 포함하는 파일
      • gnb.html - [div id="skip"], [header] 영역소스 포함하는 파일
      • head.html - [head] 태그 영역소스 포함하는 파일

    • whole - 구조별로 분해되지 않은 파일별 전체 소스

  • images - image 파일폴더

  • js - javascript 파일폴더
